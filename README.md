# FlagFinder

FlagFinder is a golf course ranking app inspired by Beli. Search and discover 
golf courses near you, rate them on a 1–10 scale with a custom S through F tier 
system, log rounds with stroke counts and photos, and save courses to a personal 
bucket list for future play.

## Technologies Used
- Firebase Authentication — user login and sign up
- CloudFirestore — real-time data storage for ratings and bucket list
- Firebase Storage — photo uploads for each round
- MapKit & MKLocalSearch — course discovery and map display
- CoreLocation — device location for nearby course search
- PhotosUI — camera roll access for adding photos
- @Observable & async/await — modern Swift concurrency
- MVVM architecture — Models, ViewModels, and Views folders
- Custom S/A/B/C/D/F tier rating system
- Codable structs with @DocumentID for Firestore
